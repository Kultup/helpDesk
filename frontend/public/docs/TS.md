# Технічне Завдання (ТЗ) — Help Desk

## Огляд Проєкту

Ця система є повноцінною **Help Desk** платформою, розробленою для автоматизації обробки заявок в технічну підтримку з глибокою інтеграцією в **Telegram**.

## Основні Функції

### 1. Управління Тікетами

- Створення, редагування та закриття заявок.
- Призначення відповідальних осіб.
- Відстеження статусу та пріоритету.
- Система коментарів та вкладень.

### 2. Інтеграція з Telegram

- Автоматична реєстрація користувачів через бот.
- Сповіщення про нові тікети та зміну статусів.
- AI-асистент для швидкої відповіді на типові запитання.
- Отримання GIF-оцінок після закриття тікета.

### 3. Адміністрування

- Гнучка система ролей (Admin, User).
- Довідники: Міста, Посади, Заклади, Обладнання.
- Аналітика та звіти по завантаженості.
- Логування системних подій.

## Технологічний Стек

### Frontend

- **React 19** + **TypeScript**
- **Tailwind CSS** для стилізації
- **Lucide React** для іконок
- **i18next** для багатомовності

### Backend

- **Node.js** + **Express**
- **MongoDB** (через Mongoose)
- **Socket.io** для real-time оновлень
- Інтеграція з **Active Directory** та **Zabbix**

## Плани на Майбутнє

- Розширення бази знань.
- Глибша аналітика SLA.
- Мобільний додаток для інженерів.
